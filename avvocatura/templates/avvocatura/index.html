Sistema per la documentazione delle dipendenze tra i servizi di Avvocatura

Selezionando uno dei sistemi della lista verrà visualizzato un grafico che ne mostra le dipendenze ed una descrizione

I sistemi sono descritti come oggetti json all'interno di documenti MongoDB.
In particolare le dipendenze sono descritte con una lista avente come chiave 'deps' 

{
    '_id': ObjectId("5596ba3655037e04d25c24f9"),
    'IP': '10.148.5.22',
    'desc': 'Database di produzione',
    'deps': [ 'NSI', 'Avvjboss',  ],
    'name': 'dbserver.ads.it'
}
Un ulteriore livello di dipendenze può essere aggiunto, inserendo all'interno della lista un oggetto json: 
            {
	'name': 'postgres.ads.it',
	'IP': '10.148.5.100',
	'desc': 'Servers AlFresco',
	'deps': [ 'Console PEC', {
			'dbserver.ads.it': [ 'NSI', 'Avvjboss' ]
		} ]
}
